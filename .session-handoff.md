## codegen bug: 隔离到独立函数即绕过

**现象**：`StatementScanChecksum` 的 while 循环内访问 `scan.exprTokenCount` 破坏 `StatementExprCheck`

**根因**：冷编译器深层 codegen 问题（ARM64 指令正确但运行时异常），具体机制未定位

**有效 workaround**：将被访问字段隔离到独立函数
```cheng
// 主函数不访问 exprTokenCount
fn StatementScanChecksum(source: str): int32 =
    ...
    let exprPart = StatementScanChecksumExprPart(source)  // 独立函数
    return countScore + checksum + exprPart

// exprTokenCount 只在独立函数中被访问
fn StatementScanChecksumExprPart(source: str): int32 =
    while ...:
        acc = acc + scan.exprTokenCount  // 这里不触发 bug
    return acc
```
